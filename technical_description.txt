3.3.1. Обработка пропущенных значений
Методы заполнения включают:
- Статистические методы:
  * Среднее значение
  * Медиана
  * Мода
  * Линейная интерполяция
- Методы машинного обучения:
  * KNN-импутация
  * Итеративная импутация
  * EM-алгоритм
  * Линейная регрессия

Визуализация результатов импутации включает:
- Сравнение распределений до и после заполнения
- Анализ влияния метода на статистические характеристики
- Оценка качества импутации
- Интерактивные графики для сравнения результатов

3.3.2. Обработка выбросов
Реализованы различные методы обнаружения выбросов:
- IQR (Interquartile Range) - метод межквартильного размаха
- Z-score - метод стандартных отклонений
- Isolation Forest - алгоритм изоляции аномалий
- DBSCAN - кластеризация на основе плотности
- One-Class SVM - метод опорных векторов

Система автоматически определяет оптимальный метод на основе:
- Характеристик данных
- Распределения значений
- Количества выбросов
- Размерности данных

Визуализация включает:
- Box plots для анализа выбросов
- Scatter plots для многомерных данных
- Графики распределения до и после обработки
- Интерактивные графики с возможностью фильтрации

3.3.3. Нормализация и стандартизация
Реализован автоматический выбор метода масштабирования на основе:
- Распределения данных
- Наличия выбросов
- Размаха значений
- Требований к результатам

Поддерживаются различные методы масштабирования:
- MinMaxScaler - масштабирование в диапазон [0,1]
- StandardScaler - стандартизация (μ=0, σ=1)
- RobustScaler - устойчивое масштабирование
- MaxAbsScaler - масштабирование по максимальному модулю

Анализ распределений включает:
- Проверку нормальности
- Оценку асимметрии и эксцесса
- Сравнение распределений до и после масштабирования
- Визуализацию изменений

3.3.4. Кодирование категориальных данных
Реализованы различные методы кодирования:
- Label Encoding - числовое кодирование категорий
- One-Hot Encoding - бинарное представление категорий
- Binary Encoding - бинарное кодирование с оптимизацией памяти
- Target Encoding - кодирование на основе целевой переменной

Система автоматически выбирает оптимальный метод на основе:
- Количества уникальных значений
- Кардинальности признака
- Связи с целевой переменной
- Требований к памяти и производительности

Особенности реализации:
- Эффективная обработка больших наборов данных
- Оптимизация использования памяти
- Поддержка различных типов категориальных данных
- Визуализация результатов кодирования
